## ✨ Features

- **🎭 Authentic Voices**: Each cosmic being speaks with unique mannerisms, greetings, and wisdom
- **⚡ Real-time Streaming**: Character-by-character response animation with voice wave visualization
- **🤝 Full Council Mode**: Engage all fourteen entities simultaneously for unified guidance
- **💾 Persistent Memory**: Save and load chat sessions across the fractal timeline
- **📱 Mobile-First**: Optimized for Android with touch gestures and responsive design
- **🌌 Ethereal Aesthetic**: Glassmorphism, particle fields, and silver-bloom color palette

## 🚀 Quick Start

### Live Demo
Visit the deployed app: **https://mnlawrence3-byte.github.io/nexus-cosmic-council**

### Local Usage
Simply open `index.html` in any modern browser—no build process required.

```bash
# Clone the repository
git clone https://github.com/yourusername/nexus-cosmic-council.git

# Navigate to directory
cd nexus-cosmic-council

# Open in browser (macOS)
open index.html

# Or (Linux)
xdg-open index.html

# Or (Windows)
start index.html
